Required libs:
  <ul>
    <li>libdbus</li>
    <li>dbus-glib</li> 
    <li>glib</li>
  </ul>

Install:
    
    make
    
Use:
  <ol>
  <li>./dbus-server </li>**
  <li>./dbus-client</li>
  </ol>
